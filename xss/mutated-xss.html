The noscript tag will be interpreted differently when JavaScript is enabled or disabled, based on the HTML specification.

When JavaScript is disabled, the browser interprets differently when assigned to the innerHtml property of the template element.

When JavaScript is enabled, the code is assigned to the innerHtml property of the div element.
